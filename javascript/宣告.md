# 迴圈內的宣告(Declare variables on for loop)


```
for (let i = 0; i < 10; i++) {
  // const aaa = "hi";  // not defined
  // let aaa = "hi";    // not defined
  // var aaa = "hi";    // hi
  aaa = "hi"; // hi
}

console.log(aaa);
```
<br>

python 會變成 全域變數
```
// python 會變成 全域變數
for i in range(10):
  a = 'hi'

print(a)  // hi
```
