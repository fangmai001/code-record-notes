# Hash

each

```ruby
hash = {apple: "pen", pen: "pineapple"}

hash.each do |key, value|
    puts "Key: #{key}, Value: #{value}"
end
```

each_key

```ruby
hash = {apple: "pen", pen: "pineapple"}

hash.each_key do |key|
    puts key
end
```

each_value

```ruby
hash.each_value do |value|
    puts value
end
```
